# Samsung Supply Chain Dashboard

## 📌 Project Overview

The Samsung Supply Chain Dashboard is an interactive Power BI project designed to analyze and monitor supply chain operations. The dashboard provides insights into inventory management, sales performance, supplier efficiency, and shipment tracking, helping organizations make data-driven decisions and improve operational efficiency.

---

## 🎯 Objectives

* Monitor inventory levels and stock availability.
* Analyze product sales and demand trends.
* Evaluate supplier performance and delivery efficiency.
* Track shipment status and logistics operations.
* Identify supply chain bottlenecks and delays.
* Improve decision-making through data visualization.

---

## 🛠️ Tools & Technologies Used

* **Power BI** – Data visualization and dashboard development.
* **Microsoft Excel** – Data storage, cleaning, and preprocessing.
* **Power Query** – Data transformation and ETL processes.
* **DAX (Data Analysis Expressions)** – KPI calculations and custom measures.

5. Developed DAX measures for business KPIs.
