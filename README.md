# samsung_supply_chain
Samsung Supply Chain Dashboard
Project Description

Developed an interactive Supply Chain Dashboard using Power BI to analyze Samsung's supply chain operations. The project focuses on tracking inventory levels, product sales, supplier performance, shipment status, and logistics efficiency. The dashboard helps stakeholders make data-driven decisions by providing clear visual insights into supply chain activities.

Objectives
Monitor inventory availability across products.
Track sales performance and demand trends.
Analyze supplier delivery performance.
Monitor shipment status and logistics operations.
Identify supply chain bottlenecks and delays.
Support business decision-making through data visualization.
Tools & Technologies Used
Power BI – Dashboard creation and data visualization.
Microsoft Excel – Data cleaning and preprocessing.
Power Query – Data transformation and preparation.
DAX (Data Analysis Expressions) – Calculated measures and KPIs.

The dashboard provides a comprehensive view of Samsung's supply chain performance, enabling better inventory management, supplier evaluation, and logistics planning. It helps businesses reduce delays, optimize stock levels, and improve operational efficiency.
